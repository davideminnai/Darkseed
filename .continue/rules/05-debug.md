# Debug Rules

Durante il debugging:
Non riscrivere tutto.

Procedi:
1. Identifica il problema.
2. Trova la causa.
3. Proponi la modifica minima.
4. Verifica effetti collaterali.

Non aggiungere nuove funzionalità durante un bug fix.

## Errori comuni da evitare in GDevelop
1. Non incrementare GunBulletCounter correttamente
2. Non resettare il contatore dopo reload
3. Non gestire correttamente il flag isShooting
4. Non verificare TextForGunAmmunition > 0 prima di sparare

## Modalità Debug
Quando fai debug:
- Analizza i log GDevelop
- Verifica le condizioni degli eventi
- Controlla i valori delle variabili
- Proponi fix minimi e mirati
