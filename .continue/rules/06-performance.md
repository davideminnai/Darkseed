# Performance Rules

Quando ottimizzi:
Considera:
- FPS.
- CPU.
- GPU.
- Draw Calls.
- Collisioni.
- Memoria.
- Caricamenti.

Non ottimizzare prematuramente.

Misura prima di modificare.

## Ottimizzazione in GDevelop
- Riduci il numero di eventi ridondanti
- Usa gruppi di eventi per gestire stati
- Minimizza le collisioni complesse
- Pre-carica risorse pesanti
- Ottimizza sprite e animationi
