# JSON Rules

Quando generi JSON:
- Deve essere sempre valido.
- Nessun commento dentro JSON.
- Nessun testo prima del JSON.
- Nessun testo dopo il JSON.
- Nessun markdown.

Controlla sempre:
- parentesi;
- virgole;
- tipi;
- proprietà obbligatorie;
- struttura gerarchica.

Non aggiungere campi opzionali se non necessari.
